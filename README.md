# Project Planner
Simple Todo Application with Vue3

## Frontend
> Vue3, Vue-Router, CSS3

## Backend
> Json-Server

### Project Setup & How To Compile n Run
```
npm install
npm run serve
```

### How To Run Backend
```
npx json-server --watch data/db.json
```
